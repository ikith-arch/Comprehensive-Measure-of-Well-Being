# 🌍 HDI Prediction System using Machine Learning

## 📌 Project Overview

The **Human Development Index (HDI) Prediction System** is a Machine Learning based web application that predicts the Human Development Index value of different countries using socio-economic factors.

The system analyzes country-level development indicators such as education, health, income, internet usage, employment and population to estimate HDI.

The application provides:
- HDI prediction
- Data visualization dashboard
- Statistical analysis
- Country-wise development insights
- PDF report generation


---

# 🚀 Features

## 🔍 HDI Prediction

- Predicts HDI value for selected countries
- Classifies development level:
  - Very High Human Development
  - High Human Development
  - Medium Human Development
  - Low Human Development


## 📊 Analytics Dashboard

Dashboard provides:

- Total number of countries
- Average HDI score
- Highest HDI country
- Lowest HDI country
- HDI visualization charts


## 📈 Data Visualization

Includes:

- Top 10 Countries by HDI
- HDI Distribution Graph
- Feature Correlation Heatmap


## 📄 PDF Report

Users can generate and download prediction reports in PDF format.

# ⚙ Installation and Setup

## 1. Clone Repository

```
git clone <repository-link>
```

## 2. Open Project Folder

```
cd HDI-Prediction-System
```

## 3. Install Required Libraries

```
pip install -r requirements.txt
```

## 4. Run Application

```
python app.py
```

## 5. Open Browser

```
http://127.0.0.1:5000/
```
