🌍 About HDI Prediction System


The Human Development Index (HDI) Prediction System is a Machine Learning based web application that analyzes country-level socio-economic factors and predicts the Human Development Index value.


🚀 Project Features


🤖 Machine Learning Model Linear Regression
📊 Dataset Country-wise HDI Information
🌍 Prediction Human Development Index Estimation
📈 Dashboard Interactive Data Visualization
🛠 Technologies Python, Flask, Pandas, Scikit-Learn
